# DDP_ui_HCC
A initial frontend for visualizing the HCC dataset made using Niivue.

Currently only works with nifti files.

Have the filenames in this format.
Base Image -- {case_id}.nii.gz"
Ground Truth -- {case_id}_gt.nii.gz
Prediction -- {case_id}_pred.nii.gz"

Run these three - npm install chart.js, npm install @niivue/nivvue @niivue/dicom-loader, npm install react-router-dom
